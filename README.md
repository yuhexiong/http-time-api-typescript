# http time API
A server can see time in timezone.  

## Overview

- Language: TypeScript
- Web FrameWork: Express

## Install modules

```bash
npm install
```

## Build

```bash
tsc
```

## Run

```bash
nodemon build/server.js
```

## API

* `GET /time`
* `POST /time`
* `PUT /timezone`
* `DELETE /timezone`
